CTE Esport Map Interactive Map Plugin Introduction

CTE Esport Map 是一款专为 SillyTavern 打造的沉浸式互动地图插件，旨在为电竞战队助理故事提供极具代入感的可视化舞台。插件内置根据世界书描述的周边环境 1:1 还原的战队基地周边高清大地图，囊括基地内部、万城娱乐、商业广场、小吃街、电竞馆等十余个关键地标。
CTE Esport Map is an immersive interactive map plugin exclusively crafted for SillyTavern, designed to provide an incredibly engaging visual stage for esports team assistant narratives. The plugin features a high-definition, 1:1-scale map of the team base's surroundings, meticulously recreated based on the World Book descriptions. It encompasses over ten key landmarks, including the base interior, Wancheng Entertainment, Commercial Plaza, Snack Street, and the Esports Arena.

:sparkles: 核心亮点 Core Highlights：

:rocket: 智能出行系统：点击任意地点即可唤起交互弹窗，支持选择“独自前往”或“邀请某人”。系统会自动生成如 {{user}} 邀请{{char}}前往{{location}}的剧情指令并填入对话框，一键引导AI生成后续故事剧情。
Deep Scene Interaction: Click any location to activate an interactive pop-up window, supporting options to “Go Alone” or “Invite Someone.” The system automatically generates story prompts like {{user}} invites {{char}} to {{location}}, filling the dialogue box to instantly guide AI-generated narrative progression.

:office: 深度场景交互：特设基地内部档案视图，采用全新的垂直布局，支持查看详细的楼层平面图，可精确探索从顶楼天台到一楼大厅的每一个房间。
Deep Scene Interaction: Features a dedicated internal archive view with a new vertical layout, enabling detailed floor plan exploration—from rooftop terraces to ground-floor lobbies.

:art: 高度自由定制 Advanced Customization：

自定义背景：允许用户上传本地图片替换原始地图，并提供一键恢复默认功能。
Custom Backgrounds: Upload local images to replace default maps with one-click default restoration.

自由布局：支持拖拽地标，用户可根据自定义地图随意调整地点位置。
Free Layout: Drag-and-drop landmarks to freely reposition locations on custom maps.

多主题切换：内置黑金、蓝白、粉白三款 UI 主题。
Multi-Theme Switching: Built-in UI themes: Black & Gold, Blue & White, Pink & White.
