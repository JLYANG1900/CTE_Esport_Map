:trophy: CTE Esport Map 互动地图插件介绍

CTE Esport Map 是一款专为 SillyTavern 打造的沉浸式互动地图插件，旨在为电竞战队助理故事提供极具代入感的可视化舞台。插件内置根据世界书描述的周边环境 1:1 还原的战队基地周边高清大地图，囊括基地内部、万城娱乐、商业广场、小吃街、电竞馆等十余个关键地标。

:sparkles: 核心亮点

:rocket: 智能出行系统：点击任意地点即可唤起交互弹窗，支持选择**“独自前往”或“邀请某人”**。系统会自动生成如 {{user}} 邀请 秦述 前往万城娱乐总部 的剧情指令并填入对话框，无缝衔接 RP 体验。

:office: 深度场景交互：特设基地内部档案视图，采用全新的垂直布局，支持查看详细的楼层平面图，可精确探索从顶楼天台到一楼大厅的每一个房间。

:art: 高度自由定制：

自定义背景：允许用户上传本地图片替换原始地图，并提供一键恢复默认功能。

自由布局：支持拖拽地标，用户可根据自定义地图随意调整地点位置。

多主题切换：内置黑金、蓝白、粉白三款 UI 主题。
